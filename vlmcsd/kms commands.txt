WINDOWS 8.1
===========
verbose output of the Software Licensing service
slmgr.vbs /dlv
cscript c:\Windows\System32\slmgr.vbs /dlv all  (terminal output)

Manually assign a KMS host to KMS clients by using KMS host caching
slmgr.vbs /skms <value>:<port>

Manually activate KMS clients
slmgr.vbs /ato

View license expiration
slmgr /xpr
